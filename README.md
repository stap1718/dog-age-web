# Learning Objectives

This project was created to practice:

- Flask web development
- HTML forms
- Input validation
- Python functions
- Converting desktop GUIs to web apps

# Challenges Encountered

- Learning how Flask routes work
- Connecting Python to HTML forms
- Styling the webpage with CSS
- Handling form validation

# Reflection

This project helped me understand how desktop applications can be redesigned as web applications. I learned how Flask handles user input and how HTML/CSS replaces tkinter widgets in a browser environment.

# Converting from tkinter to Flask

This project originally existed as a desktop GUI application using tkinter. During the conversion process:

- tkinter Entry widgets became HTML input fields
- tkinter Buttons became HTML buttons
- Python GUI output became dynamic webpage content
- CSS replaced tkinter styling
